Design a portfolio screen in Figma based on content from Notion.

Arguments: $ARGUMENTS (screen name + Notion URL or description)

1. Read the relevant content from Notion (via Notion MCP)
2. Plan the screen layout: sections, hierarchy, content blocks
3. Follow the `design-in-figma` skill for Figma Plugin API patterns
4. Follow the `design-system` skill for colors, typography, spacing
5. Create the frame at 1440px width via `figma_execute`
6. Take a screenshot via `figma_take_screenshot` to verify
7. Report what was created and what Roberto should refine manually
